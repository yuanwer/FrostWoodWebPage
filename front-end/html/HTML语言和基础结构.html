#HTML 语言和基础结构

HTML（HyperText Markup Language），超文本标记语言，是用于创建网页的一种标记语言。

那什么是标记语言呢？

  • 由若干个标记或者叫标签（tag），组成的语言，就叫做标记语言；
  • 标签的作用就是对某些内容进行标记，让浏览器能够识别和解析（比如 h1 标签，浏览器就知道这是一个标题）；
  • 标签和标签包裹的内容，共同组成的部分，叫做元素（element）。

那什么是超文本呢？超文本的意思就是超越文本，不仅仅只有文字，还可以有图片，视频，音频，链接等。

#HTML 文件的后缀名

HTML 文件的后缀名是 .html 或者 .htm。

为什么有两个呢？首先，现代前端开发统一使用 .html 作为文件后缀名， 而 .htm 的存在，是由于早期的 Windows 操作系统，对文件的后缀名有长度限制，不能超过 3 个字符，所以才使用了 .htm 作为文件后缀名。

#HTML 的结构

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <title>Document</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

上面的代码，就是 HTML 文件的结构。主要包含以下几个部分：

  • <!DOCTYPE html>：声明文档类型，告诉浏览器这是一个 HTML5 文档；
  • <html>：根元素，所有的 HTML 代码都包含在这个标签中；
  • <head>：头元素，包含文档的元数据，比如字符集，标题等；
  • <title>：标题元素，定义文档的标题；
  • <body>：主体元素，包含文档的主体内容，也就是我们看到的页面内容。

#元素

上面已经说了，元素（element）是由标签（tag）和内容（content）组成的，元素是网页的一部分。

一个元素，可以是一段文本，或者是一个图片，或者是一个空的标签。

实际开发中，能使用到的元素，其实并不多，但是所有元素总共的数量，还是非常多的， 具体可以参考：HTML 元素参考手册。

这些元素不需要去记忆，常用的元素多用几次自己就记住了，那些不常用的元素。可以去上面的手册里面查找。

#元素的构成

<h1>Hello World</h1>

<h1> 是开始标签，</h1> 是结束标签，Hello World 是内容，他们三个一起，组成了一个元素。

#单标签元素和双标签元素

双标签元素，就是既有开始标签，又有结束标签的元素。比如：

<h1>Hello World</h1>
<p>Hello World</p>
<div>Hello World</div>
<html></html>
<body></body>

双标签元素是因为它们元素的开始和结束标签之间需要放置内容。

单标签元素，就是只有开始标签，没有结束标签的元素。比如：

<br>
<hr>
<img>
<input>
<link>
<meta>

单标签元素是因为它们元素的开始和结束标签之间不需要放置内容。 比如 <img> 标签，它表示一张图片，但是图片是使用 src 属性来设置的，而不是放在标签之间。

单标签元素有 3 种写法：

<!-- 以 / 结尾 -->
<img/>
<!-- 以 / 结尾，/ 前面有空格 -->
<img />
<!-- 没有结束标签 -->
<img>

前两种写法，是从旧时代（HTML4 和 XHTML）遗留下来的写法，现在不推荐使用。

第三种写法，是 HTML5 的写法，推荐使用。

元素可以是大写的，也可以是小写的，但是推荐使用小写，下面的代码，实际作用是一样的：

<H1>Hello World</H1>
<h1>Hello World</h1>

#元素的属性

大部分情况下，元素的属性，是用来设置额外信息的，不会出现在显示的内容中。

元素的属性需要遵循以下规则:

  1. 在元素名称和第一个属性之间必须有一个空格，多个属性之间也需要用空格分隔
  2. 属性名称后如果需要设置值，则紧跟一个等号 =
  3. 属性值必须用双引号 " 包裹
  4. 属性是写在元素的开始标签中的

举个例子：

<div id="text" class="text" style="color: red;">Hello World</div>

#元素属性的分类

元素属性的分类呢，可以分为两类。

第一类呢就是公共属性。它就表示每一个元素，它都支持这个属性。比如说 class id title。

另外一类属性呢，它叫做特有属性，只有特定的元素，才支持这个属性。比如说 meta 标签的 charset 属性。

#元素的嵌套

某些元素的内容，除了可以包含文本，还可以包含其他元素。也就是元素可以嵌套。

比如：

<div>
  <h1>Hello World</h1>
</div>

元素之间的关系，有父子关系、兄弟关系。

父子关系：

下面的代码中，<div> 是父元素，<h1> 是子元素。

<div>
  <h1>Hello World</h1>
</div>

兄弟关系：

下面的代码中，<div> 和 <div> 是兄弟元素。

<div>
  <h1>Hello World</h1>
</div>
<div>
  <h1>Hello World</h1>
</div>

#HTML中的注释

注释的作用是，在代码中添加一些解释性的文字，方便开发者阅读和理解代码。

浏览器会解析注释，但是不会显示在页面上。

注释的写法是：

<!-- 单行注释 -->

<!-- 
  多行注释
  多行注释
  多行注释
-->

注释的作用主要有：

  1. 记录自己的实现思路，方便以后阅读和理解代码；
  2. 帮助别人理解代码，减少沟通成本；
  3. 开源项目中，帮助别人理解代码；
  4. 用于代码的调试；
