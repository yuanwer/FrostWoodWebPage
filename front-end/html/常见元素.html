#常见元素

#html

根元素（也叫顶级元素、根标签），表示一个文档的根，所有的 HTML 代码都包含在这个标签中。

#lang 属性

根元素上通常会设置一个 lang 属性（language），用来指定文档的语言，比如 zh-CN 表示中文，en-US 表示英文。

这个属性不是必须的，但是推荐设置，因为它可以：

  1. 帮助语音合成工具，确定发音； 语音合成工具，可以让特殊群体（比如盲人）通过语音来阅读网页内容，设置 lang 属性可以让语音合成工具根据语言来发音。
  2. 帮助翻译工具，确定翻译语言； 当你想要阅读一个英文网页时，该网页的 lang 属性是 en-US，翻译工具就会知道这个网页是英文的，然后将英文翻译为你的语言。
  3. 帮助搜索引擎，确定搜索范围。 当你在使用搜索引擎搜索时，搜索引擎会根据你搜索的关键词的语种来搜索，比如搜索 你好，搜索的结果会优先显示 lang 属性为 zh-CN 的网页。

#head

head 元素，表示文档的头部，包含当前网页的配置信息：

  • 元数据（metadata），也就是描述数据的数据：比如字符编码、标题等
  • 样式表（stylesheet）：比如 CSS 文件
  • 脚本（script）：比如 JavaScript 文件

至少会包含两个配置：

  1. 网页的标题
<title>网页标题，会显示到浏览器的标签上</title>
  1. 字符编码
<meta charset="utf-8">

准确的设置网页的字符编码，可以避免浏览器出现乱码，让浏览器可以精准的显示网页文字。

一般会设置为 utf-8，因为它是国际通用的字符编码，可以显示世界上所有的语言。

#body

body 元素，表示文档的主体，包含网页的正文内容。

用户在浏览器网页中能看见的东西，都是编写到 body 元素中的。

#h 元素

h 元素（heading），表示标题，一共有 6 个，从 h1 到 h6，数字越大，表示标题的级别越低。

h 元素是块级元素，会独占一行，上下会自动添加一定的间距。

<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>

#p 元素

p 元素（paragraph），表示段落，包含一段文字。

p 元素是块级元素，会独占一行，上下会自动添加一定的间距。

<p>这是一个段落元素，它用于包含一段文字。</p>

#img 元素

img 元素（image），表示图片，用于在网页中插入图片。

两个常用的属性：

#src 属性

src（source）：图片的地址。

  • 可以是一个本地的图片地址
  • 也可以是一个网络的图片地址
  • 还可以是 base64 编码的图片
<!-- 网络图片 -->
<img src="https://example.com/image.jpg" alt="网络图片示例">

<!-- 本地图片 - 绝对路径 -->
<img src="D:/absolute/path/to/image.jpg" alt="本地图片绝对路径示例">

<!-- 本地图片 - 相对路径 -->
<img src="./relative/path/to/image.jpg" alt="本地图片相对路径示例">

<!-- base64 格式图片 -->
<img src="data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAUA" alt="base64 图片示例">

网络图片的地址，通常是完整的 URL，比如 https://example.com/images/picture.jpg，这是最常用的方式。

本地的图片地址，后续会和代码一起部署到服务器。

本地图片有两种方式：

  1. 绝对路径（几乎不使用）：从电脑根路径开始，一直到图片文件的路径，比如 D:/absolute/path/to/image.jpg。
  2. 相对路径（常用）：从当前文件所在的路径开始，到图片文件的路径，比如 ./relative/path/to/image.jpg。

相对路径的规则：

  • . 表示当前目录，可以省略
  • ../ 表示上一级目录
  • ../../ 表示上上一级目录

在 Web 开发中，不管是什么操作系统，路径分隔符都使用 /。

#alt 属性

alt（alternative）：图片的描述。

  • 当图片无法显示时（比如资源地址错误或者图片损坏），会显示这段文字。
  • 屏幕阅读器会读取这段文字，来帮助视力障碍者了解图片的内容。
<img src="图片地址" alt="图片描述">
WARNING

img 元素，有 width 和 height 属性，可以设置图片的宽度和高度，border 属性可以设置图片的边框。 但是现代开发已经不推荐使用这些属性了，而是推荐使用 CSS 来设置图片的样式。

#可替换元素

img 是一个典型的可替换元素。可替换元素就像是一个"占位符"，其显示的内容是由外部资源或属性决定的，而不是由标签里面的内容决定。

想象你有一个相框（这就是 HTML 元素），可替换元素就像是你可以在相框里放不同的东西：

<img src="猫咪.jpg">
<img src="狗狗.jpg">
  • 同样的 <img> 标签
  • 但通过改变 src 属性，可以显示不同的图片
  • 这就是"可替换"的意思：内容可以被替换

可替换元素与普通元素的区别：

  • 普通的 HTML 元素（如 <div>、<p>）：你写什么内容就显示什么内容
  • 可替换元素：显示的内容是由外部资源或属性决定的，而不是由标签里面的内容决定

#img 支持的格式

支持的格式非常多，常见的 jpg、png、gif、svg 都支持，具体的可以参考 MDN 文档。

#a 元素

a 元素（anchor），表示超链接，用于在网页中插入超链接，从当前网页跳转到另一个链接（可能是另一个网页、视频、文件，也可能是当前网页的某个位置）。

#href 属性

href（hypertext reference）：指定要跳转的链接地址，可以是网络地址，也可以是本地地址。

<!-- 网络地址 -->
<a href="https://www.baidu.com">百度</a>

<!-- 本地地址 -->
<a href="./relative/path/to/image.jpg">本地图片</a>

也可以是当前网页的某个位置，比如 #id，表示跳转到当前网页的某个 id 为 id 的元素。这叫做锚点链接。

<div id="first">第一章</div>
<div id="second">第二章</div>
<div id="third">第三章</div>

<a href="#first">跳转到第一章</a>
<a href="#second">跳转到第二章</a>
<a href="#third">跳转到第三章</a>

#target 属性

target：指定超链接的打开方式，常用的有 _self（默认，在当前页面打开）、_blank（在新窗口打开）。

<!-- 在新窗口打开 -->
<a href="https://www.baidu.com" target="_blank">百度</a>

<!-- 在当前页面打开 -->
<a href="https://www.baidu.com" target="_self">百度</a>
INFO

现代浏览器上所看见的“新标签页”，实际上是一个新的窗口。
