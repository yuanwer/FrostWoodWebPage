    #文档声明

    在每个 HTML 文件内容的开头部分，总是有一个 <!DOCTYPE html> ，它叫做文档声明，作用是声明文档的类型。

    它会告诉浏览器，这个文件是一个 HTML 文件，HTML 的版本是 HTML5，让浏览器使用 HTML5 的解析规则来解析这个文件。

    必须放在 HTML 文件的第一行，否则可能出现兼容性问题，比如同样的代码，在不同的浏览器中，显示效果不一样。

    不同的 HTML 版本，文档声明的写法也是不同的：

    • HTML5：<!DOCTYPE html>
    • HTML4.01：<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
    • XHTML 1.0：<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
